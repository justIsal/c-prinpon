Pengertian,sejarah, dan cara kerja HTML

14 mei 2022

Pengertian HTML

HTML merupakan singkatan Hyper Text Markup Language adalah bahasa markup, untuk membuat struktur halaman sebuah website. HTML merupakan pondasi penting dalam pembuatan sebuah website.

sejarah HTML

HTML di ciptakan oleh Tim Berners-Lee merupakan seorang fisikawan di lembaga penelitian CERN,Swiss. Beliau merupakan seseorang yang berjasa bagi kita, karena beliau kita bisa mengakses website setiap hari, pada tahun 1991, Tim merilis versi html pertama yang di dalamnya terdapat sekitar 18 tag. Bisa di lihat di link berikut … html tersebut berisi tentang panduan membuat laman web dan elemen hypertext yang digunakan dalam prosesnya. Tim bernerslee juga merupakan seorang direktur dari organisasi World Wide Web Consortium(W3C) di massachusetts Institute Of technology. W3C merupakan sebuah organisasi yang mengakomodir aturan-aturan yang di web browser, seperti pegembangan html hingga saat ini HTML 5, mengalami perubahan yang signifikan seperti dan terdapat fitur semantic untuk memudahkan developer, seperti <article>, <footer>, <header>, dan <main>.

Fungsi

HTML berfungsi untuk bagaimana cara menampilkan sebuah kontent ke dalam sebuah browser, HTML sering di kolaborasikan dengan penggunaan CSS (Casecade Style Sheet) serta javascript. Dimana HTML berfungsi untuk membuat kerangka dan struktur halaman sebuah website, CSS berfungsi untuk memberikan tampilan desain seperti perubahan warna, font, mengatur margin dan lain sebagainya, dan JavaScript merupakan bahasa pemrograman untuk membuat website tersebut menjadi interaktif.

kerangka Inti HTML dan cara kerja HTML

  • Tag

    • Tag merupakan perintah atau awalan instruksi yang nantinya akan di tampilkan pada web browser. Seperti <p> untuk menampilkan paragraf, <li> untuk mebuat list, <h1> untuk membuat judul, dan lain sebagainya.

      <p>ini adalah sebuah paragraf </p>
  • Element

    • Element merupakan proses keseluruhan kode di mulai dari tag pembuka (<>) lalu isi kontent, dan di akhiri tag penutup (</>), sebagai contoh :

  • Attribute

    • Tag dapat mempunyai atribute. Atribute merupakan perintah tambahan yang di masukan ke dalam sebuah elemen. Seperti contoh tag <img> ini tidak mempunyai atribute, tetapi kalau mempunyai atribute <img src=”gambar.jpg”/>, atributnya adalah src.

kerangka HTML

  1. <!DOCTYPE html>

    2. Berfungsi untuk mendeklarasikan kepada komputer bahwa anda menuliskan sebuah perintah di dalam kode HTML.

  2. <html></html>

    3. Berfungsi untuk tag yang menandakan proses mulai menuliskan kode program pada dokumen HTML.

  3. <head></head>

    4. Tag ini digunakan untuk menambahkan metadata ke dalam dokumen html yang berisi judul, deskripsi, library dan lain sebagainya.

  4. <body></body>

    5. Dan yang terakhir, tag body digunakan sebagai tempat untuk menuliskan setiap elemen atau lebih tepatnya disebut juga dengan konten pada HTML itu sendiri.

Kesimpulan

HTML merupakan bahasa markup yang berfungsi untuk bagaimana cara menampilkan sebuah kontent ke dalam sebuah browser, di dalam html juga terdapat tag, atribute, dan element, seperti yang sudah di jelaskan di atas.
